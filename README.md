# 🏨 Hotel Booking App

A full-stack **Hotel Booking Web Application** built using **Node.js, Express, MongoDB, and EJS**.  
Users can explore hotels, register/login, and manage bookings with authentication and authorization.

---

## 🚀 Features

- 👤 User Authentication (Register / Login / Logout)
- 🏨 Hotel Listings with Details
- ✏️ Add, Edit & Delete Hotels (Authorized Users)
- 🔐 Secure Authentication using Passport.js
- 💾 MongoDB Database Integration
- 🌐 RESTful Routes
- 📦 Session Management
- 🎨 Server-Side Rendering with EJS
- ⚠️ Flash Messages for User Feedback

---

## 🛠️ Tech Stack

**Frontend**
- EJS
- Bootstrap / CSS

**Backend**
- Node.js
- Express.js

**Database**
- MongoDB
- Mongoose

**Authentication**
- Passport.js
- Express-Session
